# 🌦️ Weather Dashboard - Kurulum Kılavuzu (Basit Versiyon)

## ⚡ Hızlı Başlangıç (5 saniye)

### Windows'ta:
1. `weather-dashboard` klasörünü açın
2. **`Weather_Dashboard.bat`** dosyasına **çift tıklayın** 
3. **Bitti!** Uygulama otomatik açılacak ✅

---

## 📥 İndirme Adımları

### 1. Repository'yi İndir
- Git komutunu bilmiyorsanız:
  - https://github.com/Freeman662/yukleme_optimizasyonu adresine gidin
  - **Code** (yeşil buton) → **Download ZIP** tıklayın
  - ZIP dosyasını çıkartın

### 2. Klasörü Aç
```
yukleme_optimizasyonu/
└── weather-dashboard/          ← Bu klasörü açın
    ├── Weather_Dashboard.bat   ← Buraya çift tıklayın ⭐
    ├── index.html
    ├── css/
    ├── js/
    └── ...
```

### 3. Başlat
**`Weather_Dashboard.bat`** dosyasını **çift tıkla** ve bitti! 🎉

---

## 🎯 Kullanım

### Şehir Arama
1. Üstte arama kutusuna şehir adı yazın
2. **Enter** tuşuna basın
3. Hava durumu yüklenecek

**Örnek:** `Istanbul`, `London`, `New York`

### Saatlik Hava Durumu
- Sağa kaydırarak saat bazında tahmin görebilirsiniz

### 5 Günlük Tahmin
- Aşağıya kaydırarak günlük tahmin görebilirsiniz

### GPS Konumunuzu Kullanın
- Sağ üstte 📍 simgesine tıklayın
- **İzin Ver** tıklayın
- Mevcut konumunuzun hava durumu gösterilecek

### Favorilere Ekle
- Şehir adının yanındaki ⭐ simgesine tıklayın
- Favorilere eklenir (sol tarafta görünür)

### Tema Değiştir
- Sağ üstte 🌙 simgesine tıklayın
- Dark/Light mode arasında geçiş yapabilirsiniz

### Sıcaklık Birimi
- Sol tarafta **°C** veya **°F** seçebilirsiniz

## 🚨 Sorunlar ve Çözümler

### ❌ "index.html dosyası bulunamadı"
**Çözüm:** `Weather_Dashboard.bat` dosyasıyla aynı klasörde `index.html` olmalı

### ❌ Uygulama açılmıyor
**Çözüm:** 
- Dosyaların tamamını indirdiğinizden emin olun
- İnternet bağlantınızı kontrol edin
- Tarayıcıyı kapatıp yeniden başlayın

### ❌ Hava durumu yüklenmediği
**Çözüm:**
- İnternet bağlantınızı kontrol edin
- Sayfayı yenile (F5)
- Şehir adını doğru yazın

### ❌ GPS çalışmıyor
**Çözüm:**
- Tarayıcı izinlerini kontrol edin
- GPS açık mı diye kontrol edin
- Firefox/Chrome kullanın
